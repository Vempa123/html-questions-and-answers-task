HTML Questions and Answers

What is HTML?

HTML, or HyperText Markup Language, is the standard language used to create and structure content on the web. It consists of a series of elements or tags that define the structure of a webpage.

Why do we use HTML?

We use HTML to create the structure of web pages, allowing browsers to display content such as text, images, links, and multimedia.

What is doctype in HTML?

The doctype declaration defines the document type and version of HTML being used, ensuring that browsers render the page correctly.

What is the root element of HTML?

The root element of HTML is the <html> tag, which encompasses all the content on the webpage.

What is a tag?

A tag is a markup construct that specifies the start and end of an element in HTML, typically enclosed in angle brackets, like <tagname>.

Give me a few examples of self-closing tags?

Examples of self-closing tags include <br />, <img src="image.jpg" />, and <input type="text" />.

What is an element?

An element is a complete structure in HTML that includes a start tag, content, and an end tag, such as <p>This is a paragraph.</p>.

How many types of elements are there in HTML?

There are two main types of elements in HTML: block elements (e.g., <div>, <p>) and inline elements (e.g., <span>, <a>).

What is the difference between inline and block elements?

Block elements occupy the full width available and start on a new line, while inline elements only take up as much width as necessary and do not start on a new line.

What is the difference between semantic and non-semantic elements?

Semantic elements clearly describe their meaning in a human- and machine-readable way (e.g., <header>, <article>), while non-semantic elements do not convey any specific meaning (e.g., <div>, <span>).

What are href vs src attributes?

The href attribute specifies the URL of a link, while the src attribute specifies the URL of an image or external resource.

What are the features of HTML5?

Features of HTML5 include native support for audio and video, improved parsing rules, new semantic elements (e.g., <article>, <section>), and APIs for local storage, geolocation, and more.

What are entities in HTML?

Entities are special codes used to represent characters that have a specific meaning in HTML, such as & for & and < for <.

What is div vs span?

<div> is a block-level element used for grouping content, while <span> is an inline element used for styling a portion of text within a block.

What are list tags in HTML?

List tags include <ul> for unordered lists, <ol> for ordered lists, and <li> for list items.

What are attributes?

Attributes provide additional information about an HTML element and are defined in the start tag, typically in name/value pairs, like name="value".

Mention a few attributes for img tags?

Attributes for <img> tags include src (source of the image), alt (alternative text), and width and height (dimensions of the image).

What is <thead> in a table?

<thead> is used to group the header content in an HTML table, typically containing <tr> (table row) elements.

What is <tr>?

<tr> stands for table row and is used to define a row within a table.

What is <th>?

<th> stands for table header and is used to define a header cell in a table row.

What is <td>?

<td> stands for table data and is used to define a standard cell in a table row.

Can I use <th> in <tbody>?

Yes, you can use <th> elements within <tbody> for cells that serve as headers for the data in that section.

What are meta tags?

Meta tags provide metadata about the HTML document, such as character set, author, viewport settings, and descriptions for search engines.

What is SEO?

SEO, or Search Engine Optimization, is the practice of enhancing a website to improve its visibility in search engine results, aiming to attract more organic traffic.

What is the attribute of the HTML tag?

The HTML tag itself can have attributes such as lang (to specify the language of the document) and dir (to specify the text direction).
